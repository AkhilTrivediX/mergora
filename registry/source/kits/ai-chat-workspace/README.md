# AI Chat Workspace kit

This source-only workflow composes canonical public Mergora components for conversation navigation, provider-neutral streaming messages, attachments, citations, tool states, retry, edit, branch, offline, error and empty paths.

`AiChatWorkspaceAdapter` keeps model, network, storage and tool execution consumer-owned. `createDeterministicAiChatWorkspaceAdapter` is a local fake with fixed seed data and no network or credential requirement. Follow-output, announcements, attachments, suggestions, role context, streaming cursor, citation detail, tool detail, character budget and the composer shortcut remain separate opt-ins. The basic workspace adds none of their UI, behavior, events or accessibility output.

Installation does not complete a backend, security boundary, privacy program, legal/model-safety review, moderation system, data-retention policy, or product-level accessibility assessment.

Status: source present and unreleased. Generated parity, clean-consumer install/update/remove, packed consumers, full workflow/browser evidence, and current manual security/privacy/legal/assistive-technology records remain blockers.
