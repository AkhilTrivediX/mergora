# Scheduler Kit

Explicitly Beta source-owned calendar/agenda workflow with calendar filters, event creation/editing, timezone selection, conflict recovery, keyboard calendar navigation, and a narrow-screen agenda alternative.

## Mergora advantage

Timezone context, duration summaries, collision guidance, and polite change announcements are independent opt-ins. Each removes its UI, behavior, events, and accessibility output when disabled; the basic calendar, filters, agenda, and native form remain predictable.

## Integration boundary

Consumers own wall-time-to-instant conversion, recurrence, authorization, persistence, notification delivery, attendee privacy, and legal policy. The fictional deterministic adapter accepts UTC only.

## Status

`source-present-unreleased` and explicitly `beta` target maturity. No Stable claim is permitted without every required promotion gate.
