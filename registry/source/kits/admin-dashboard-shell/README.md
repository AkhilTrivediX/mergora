# Admin Dashboard Shell

This source-only workflow composes canonical Mergora navigation, breadcrumb, chart, data-table, status, and recovery components around an abort-aware consumer adapter. It includes responsive navbar and sidebar paths, a chart with a real table fallback, dense activity data, optional notifications, and a deterministic fictional fixture.

## Mergora advantage

Role-aware navigation can remove inaccessible destinations while a visible context note keeps the authorization boundary honest. Notification actions, interactive chart points, an always-visible chart table, and activity query tools are separate opt-ins. Disabling each enhancement removes its UI, events, and accessibility output while the literal-white navigation, chart fallback, and readable table remain usable.
