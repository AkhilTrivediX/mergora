# Settings Workspace

This source-only workflow provides required profile, preferences, notifications, and security sections; controlled or uncontrolled section and dirty state; native save and reset; async recovery; and consumer-rendered canonical controls. Repeated instances own unique dialog labels, read-only reset is inert, and unsaved prompts restore focus to the attempted navigation control. Persistence, authorization, notification delivery, account operations, and network behavior remain consumer-owned.

## Mergora advantage

Optional unsaved-change protection intercepts only risky section and page exits and restores deliberate keyboard focus. A separate optional destructive action requires review and exact typed confirmation. Disabling either feature removes its listeners, interception, state, controls, callbacks, and accessibility output while normal section navigation and saving remain unchanged.
