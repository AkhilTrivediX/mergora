# DatePicker canonical source

Status: source present and unreleased. No Stable or parity claim is made.

`DatePicker` keeps native date entry and mobile picker behavior while optionally adding consumer-owned date presets. `presets={false}` is the default and removes preset UI, behavior, events, and accessibility output. `showDateContext` remains separately configurable through the underlying Mergora field.

The precise Canvas field, Ink boundary, restrained preset controls, Green context rail, and Violet focus seam use shared semantic tokens. Promotion remains blocked on generated parity, packed consumers, browser matrices, overlay/calendar integration evidence, and manual assistive-technology records.
