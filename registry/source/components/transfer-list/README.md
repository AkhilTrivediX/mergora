# Transfer List

A native dual-list transfer workbench with explicit move actions, controlled/uncontrolled membership, repeated-value serialization, reset behavior, and responsive logical reflow.

## Mergora advantage

`showTransferSummary` exposes source/destination counts and the most recent move in a visible polite rail. False prevents summary UI, live output, status work, events, and accessibility relationships.

Every collection exposes a visible count. `filterable` adds independent source/destination search and can be removed without changing membership behavior. Native multi-selection supports explicit move buttons plus `Alt+ArrowRight`/`Alt+ArrowLeft`, and the single-column narrow layout remains a complete mobile alternative.
