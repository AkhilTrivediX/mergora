# Stepper canonical source

Status: source present and unreleased. No Stable, workflow-validation, package-parity, or manual assistive-technology claim is made.

`Stepper` exposes an ordered process with one `aria-current="step"`, linear or nonlinear reachability, disabled/error state, optional navigation, and controlled or uncontrolled current value. At narrow widths the same ordered structure reflows vertically without losing labels or sequence.

Two enhancements are independent: `showProgressBar` adds native progress semantics, while `renderProgressSummary` adds consumer-localized recovery context and `announceStepChanges` adds a polite change announcement. Disabling each removes its own UI and accessibility output without affecting selection callbacks. Literal Canvas, strong Ink connectors, Green completion, Violet focus, restrained geometry, RTL-safe layout, and forced-color mappings provide the Mergora identity.

Promotion still requires generated parity, packed consumers, workflow integration evidence, and current manual assistive-technology sessions.
