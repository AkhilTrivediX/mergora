# StreamingText

`StreamingText` renders plain, selectable provider-neutral output as stable keyed segments. Appending a segment does not replace prior DOM.

## Mergora advantage

`announceUpdates` adds a debounced polite channel only when requested; `showCursor` is independent and reduced-motion safe. Disabling either removes its own UI and accessibility behavior without affecting readable content.

Status: source present and unreleased. Full generated parity, browsers, packed consumers, announcement-rate review, and manual assistive-technology evidence remain blockers.
