Status: source present and unreleased. No Stable, route-framework, package-parity, or manual assistive-technology claim is made.

`Tour` is a non-modal, skippable guidance region: it never adds a backdrop, inert state, focus trap, or blocked application access. Steps support controlled or uncontrolled progress, anchored target measurement, explicit preserve/panel focus policy, Escape dismissal, trigger or consumer-ref focus return, completion/skip callbacks, and an optional consumer route adapter.

Literal Canvas, Ink boundary, Green action, Violet progress/focus, restrained corners, anchored spatial motion, and an instant reduced-motion fallback provide the Mergora signature.
