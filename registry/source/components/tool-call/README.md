# ToolCall

`ToolCall` presents provider-neutral pending, running, success, error and cancelled states. It never executes a tool; cancellation and retry are consumer callbacks.

## Mergora advantage

Input/output disclosure, exact redaction, sensitive reveal and duration context are independently selected. Details default to absent, sensitive values default to withheld, and disabling each enhancement removes its UI, events and accessibility output.

Status: source present and unreleased. Full parity, browser, packed-consumer, security review, and current manual assistive-technology evidence remain blockers.
