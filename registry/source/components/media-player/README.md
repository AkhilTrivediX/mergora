# Media Player

Status: source present and unreleased. No Stable, caption-quality, media-security, package-parity, or manual Risk Class 3 claim is made.

`MediaPlayer` is native-first: it renders named `<audio>` or `<video controls>` and deliberately excludes autoplay from its public media props. Mergora adds independently optional transcript disclosure, chapter navigation, bounded elapsed-time announcements, tracks, and a source policy applied consistently to media, poster, and track URLs. Turning an enhancement off removes its navigation, live output, events, and accessibility relationship.

Consumers own accurate synchronized captions, audio description, corrected transcripts, URL authorization, privacy/tracking, CORS, DRM, range transport, error recovery, and media retention. A track element alone is not proof of caption quality.

The literal Canvas shell, Ink title rule, strong label hierarchy, Green playback evidence, restrained Violet chapter times, shared focus geometry, bounded media corners, logical text alignment, forced-color transcript structure, and lack of decorative motion make native controls feel related without skinning them into a brittle custom player.
