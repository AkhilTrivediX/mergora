# TableOfContents canonical source

Status: source present and unreleased. No Stable, scroll-spy accuracy, package-parity, or manual assistive-technology claim is made.

`TableOfContents` renders a named navigation landmark, ordered heading links, hierarchy depth, one current location, safe hashes, and controlled or uncontrolled current state. `collectTableOfContentsItems` is an explicit consumer-run DOM adapter; server and content-framework ownership remain outside the component.

Optional intersection observation updates only uncontrolled state and is absent when `observeCurrent={false}`. An optional current-section formatter emits concise reading context only when supplied. Disabling either removes its observer, formatter calls, output, and accessibility contribution. Literal Canvas, strong Ink rule, Green location, Violet focus, restrained corners, logical indentation, forced colors, and narrow wrapping repeat the Mergora identity.

Promotion still requires generated parity, packed consumers, long-document/scroll-container evidence, and current manual assistive-technology records.
