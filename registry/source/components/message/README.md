# Message

`Message` renders provider-neutral user, assistant, system, or tool content as a native article with explicit metadata and action regions. It never parses untrusted markup.

## Mergora advantage

`showRoleContext` adds a compact, text-backed role and delivery-state rail. Its default `false` removes the rail and all enhancement-specific accessibility output while preserving the article baseline.

Status: source present and unreleased. Generated parity, packed consumers, browser evidence, the current manual assistive-technology matrix, public dogfooding, and an approved Quality Passport remain blockers.
