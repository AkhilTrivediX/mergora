# TimePicker canonical source

Status: source present and unreleased. No Stable or parity claim is made.

`TimePicker` keeps native time editing and optionally adds an explicit collection of consumer-owned available intervals. `intervals={false}` removes the buttons, pressed states, selection callback, and accessibility group. Timezone context remains a separate opt-in feature.

Canvas surfaces, Ink boundaries, Green selection state, Violet focus/zone signals, and semantic-token geometry provide Mergora identity. Promotion remains blocked on large-interval strategy, generated parity, packed consumers, browser matrices, and manual assistive-technology records.
