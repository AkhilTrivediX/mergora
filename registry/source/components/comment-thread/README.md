# CommentThread

`CommentThread` coordinates visible author/time/body/actions, controlled or uncontrolled resolve state, and a consumer-owned reply callback.

## Mergora advantage

`optimisticReplies` adds honest pending, failed, retry and discard recovery without inventing success. `mentionSuggestions` is independent. Both default off, removing their state, handlers, UI and accessibility output.

Status: source present and unreleased. Full parity, browser, packed-consumer, mutation/security review, and current manual assistive-technology evidence remain blockers.
