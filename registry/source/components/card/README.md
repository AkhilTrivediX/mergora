# Card

Native-first Card source with Mergora's literal-white surface, ink structure, restrained green/violet state signals, bounded corners, and forced-colors treatment.

## Mergora advantage

An optional structural status rail adds contextual state without turning the whole surface into a control. Omitting or disabling the documented enhancement removes its UI, behavior, callbacks, and accessibility output while the baseline remains usable.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, supported-browser evidence, visual review, the complete Risk Class 1 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
