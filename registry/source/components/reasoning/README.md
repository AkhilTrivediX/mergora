# Reasoning

`Reasoning` is a native disclosure for a consumer-approved process summary. It does not expose or request hidden chain-of-thought.

## Mergora advantage

Bounded progress and a completion announcement are independent opt-ins. Default rendering has no automatic live-region verbosity; disabling either enhancement removes its UI and accessibility behavior completely.

Status: source present and unreleased. Full parity, browser, packed-consumer, privacy review, and manual screen-reader evidence remain blockers.
