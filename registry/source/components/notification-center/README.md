# Notification Center

Accessible read/unread notification stream with filters, loading/error recovery and controlled or uncontrolled read state.

## Mergora advantage

Queued live updates preserve reading position until the user explicitly reveals them. Grouping, bulk actions, virtualization, and read-change announcements are separate opt-ins; disabling each removes its UI, state behavior, callbacks and accessibility output.
