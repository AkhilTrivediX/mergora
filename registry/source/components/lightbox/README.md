# Lightbox

A modal image gallery with Mergora's literal-white inspection stage, strong Ink frame, precise caption hierarchy, Green position signal, Violet focus seam, restrained corners, and preference-safe spatial motion.

## Mergora advantage

Independent position context, bounded zoom with non-gesture pan controls, and optional swipe navigation add useful inspection tools while every keyboard and button path remains present. Position summary, zoom controls, and swipe navigation are separate booleans. Each false value removes its UI, state, handlers, events, and accessibility output independently.

Current status is `source-present-unreleased`. Generated parity, packed consumers, the complete Risk Class 3 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
