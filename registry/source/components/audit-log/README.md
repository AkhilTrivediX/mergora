# AuditLog

`AuditLog` presents immutable actor/action/object/time records in an ordered semantic view. Backend integrity, retention, authorization and legal policy remain consumer responsibilities.

## Mergora advantage

Filtering, fixed-row virtualization and spreadsheet-injection-resistant CSV export are independent opt-ins. Export is returned only through a consumer callback. `false` removes each enhancement's controls, work, events and accessibility output.

Status: source present and unreleased. Full parity, large-data/browser evidence, packed consumers, security/privacy review, and the manual assistive-technology matrix remain blockers.
