# Kanban canonical source

Status: source present and unreleased **Beta candidate**. Kanban is not Stable and must remain excluded from Stable counts until physical mobile assistive-technology review passes.

`Kanban` owns controlled or uncontrolled columns/cards, pointer drag, Space/Enter pickup/drop, direction-aware cross-column arrows, Escape cancellation, always-visible up/down/previous/next and destination controls, WIP rejection hooks, optional optimistic or pessimistic server persistence with rollback/retry, and consumer-controlled card windows. A visible Beta label remains in every mode.

WIP status, mobile list presentation, localized announcements, undo, server persistence, and virtualization are independent Mergora enhancements. Disabling each removes its UI, behavior, events, and accessibility output while the non-drag baseline remains operable. Literal Canvas columns, Ink boundaries, Green column seams/status, Violet grabbed/focus state, restrained corners, logical RTL layout, forced colors, and reduced-motion fallbacks provide the family identity.

Promotion beyond Beta is blocked on physical iOS VoiceOver and Android TalkBack task evidence. Generated parity, packed consumers, large-board performance, server-conflict evidence, and the remaining Risk Class 3 manual sessions are also still required.
