# Menubar

An application menubar with Mergora's literal-white command surface, flush Ink rail, strong command hierarchy, Green selection state, Violet focus seam, restrained corners, and preference-safe overlays.

## Mergora advantage

An optional associated keyboard guide and independently optional open-on-focus transfer make application-menu discovery and cross-menu exploration clearer without changing concise defaults. keyboardGuide and openMenuOnFocus are independent. Disabling the guide removes its UI and aria-describedby; disabling focus transfer removes that behavior and related open-state events.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, the complete Risk Class 3 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
