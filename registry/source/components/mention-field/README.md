# Mention Field

A native textarea with contextual mention suggestions, controlled/uncontrolled state, native form behavior, and Mergora's shared field and overlay structure.

## Mergora advantage

`showMentionSummary` makes recognized mentions inspectable in a polite visible rail. False removes the summary, live output, and description relationship while native textarea and mention insertion remain intact.

Configured trigger/entity pairs support people, topics, or other domain-neutral entities without a generic feature object. `onQueryChange` exposes typed query context for consumer-owned async suggestions, and composition events defer matching until IME input finishes. Busy and retry states are independently supplied.
