# Toast

A bounded notification queue with Mergora's literal-white surfaces, strong Ink status rail, deliberate title/context hierarchy, restrained Green/Violet/status signals, precise focus treatment, and no motion under reduced-motion.

## Mergora advantage

Urgent-priority insertion, deduplication, visible-only timers, and independently optional queue context and timer controls prevent notification floods while keeping durable recovery consumer-controlled. showQueueSummary and pauseControls are independent. False removes the corresponding UI, control behavior, events, and live-region output while automatic focus/hover timer pause and core queue behavior remain.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, the complete Risk Class 2 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
