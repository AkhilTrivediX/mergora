# BottomNavigation canonical source

Status: source present and unreleased. No Stable, package-parity, or manual assistive-technology claim is made.

`BottomNavigation` renders real destination links, one programmatic current page, safe-area padding, controlled or uncontrolled current state, disabled non-links, and consumer-owned SPA routing through cancellable link events. The default renders every destination with no disclosure.

Its optional bounded overflow keeps the current destination in the primary rail and moves only lower-priority destinations into native `details`. Setting `overflow={false}` removes the disclosure, nested list, and overflow-specific event path. Literal Canvas, Ink structure, a Green current marker, Violet focus geometry, logical layout, touch targets, RTL, forced colors, and reduced-motion-safe transitions provide the Mergora signature.

Promotion still requires regenerated distribution parity, packed consumers, cross-engine responsive evidence, and current manual keyboard, screen-reader, speech, switch, and touch records.
