# Multi Select

A searchable multi-value combobox with explicit removable tokens, controlled/uncontrolled state, repeat-name serialization, reset behavior, and Mergora selection styling.

## Mergora advantage

`showSelectionSummary` exposes selected and currently available counts in a polite visible rail. False removes the rail, live output, and description relationship without changing selection events.

`maximum` prevents additional choices without hiding selected options, while `maximumVisibleTokens` collapses only token presentation into a literal overflow count. Remote loading, retry, and pagination states remain separate and consumer-controlled.
