# FilterBuilder canonical source

Status: source present and unreleased. No Stable, package-parity, overlay-interoperability, or manual assistive-technology claim is made.

`FilterBuilder` owns controlled or uncontrolled filters, native field/operator/value editors, deterministic URL serialization, saved-filter application, form submission/reset, read-only and disabled modes, and a narrow native-dialog presentation. Every built-in string is replaceable and Unicode filter values remain data rather than markup.

Mergora's active-filter explanation, saved filters, URL adapter, and mobile drawer are independently selectable. Disabling each removes its UI, callbacks/side effects, behavior, and accessibility output without weakening the plain inline editor. Literal Canvas editors, Ink boundaries, Green status summaries, Violet filter legends/focus, restrained corners, logical layout, forced colors, and reduced-motion fallbacks provide the family identity.

Promotion still requires regenerated parity, packed consumers, nested overlay/focus evidence, URL-history evidence, and current Risk Class 3 manual desktop/mobile screen-reader, speech, switch, keyboard, touch, zoom, RTL, and forced-colors sessions.
