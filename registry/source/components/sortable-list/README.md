# SortableList canonical source

Status: source present and unreleased. No Stable, package-parity, large-data performance, or manual assistive-technology claim is made.

`SortableList` preserves ordered-list semantics and controlled or uncontrolled ownership while combining native pointer drag, Space/Enter pickup and drop, Arrow movement, Escape cancellation, always-visible Move Up/Down controls, move authorization, form serialization/reset, focus-safe scrolling, and a bounded consumer-controlled virtual window.

Destination controls, localized live announcements, undo, and virtualization are independent Mergora enhancements. Disabling each removes its UI, behavior, event path, and accessibility output without removing the non-drag baseline. Literal Canvas rows, Ink boundaries, Green committed state, Violet grabbed/focus state, restrained corners, logical narrow layout, forced colors, and reduced-motion fallbacks provide the family identity.

Promotion still requires regenerated parity, packed consumers, pointer-cancellation and large-window performance evidence, and current Risk Class 3 manual desktop/mobile screen-reader, speech, switch, keyboard, touch, zoom, RTL, and forced-colors sessions.
