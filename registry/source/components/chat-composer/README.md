# ChatComposer

`ChatComposer` is a native form with a named multiline textarea, submit/stop states, offline and recoverable error presentation. Network, upload, model and tool execution remain consumer-owned.

## Mergora advantage

IME-safe submit shortcuts, attachment presentation/removal and a character budget are separate opt-ins. Their default off paths remove related UI, handlers, events and accessibility output while native serialization and reset remain intact.

Status: source present and unreleased. Full distribution parity, browsers, packed consumers, mobile input, and manual assistive-technology evidence remain blockers.
