# Context Menu

A native context target and named menu with Mergora's literal-white overlay surface, Ink boundaries, strong menu hierarchy, Violet focus seam, restrained corners, and preference-safe motion.

## Mergora advantage

An independently optional invocation guide exposes right-click, touch/press, Shift+F10, and Menu-key discovery without altering the concise target when disabled. The optional invocation guide adds visible and programmatic discovery of every supported invocation path. Disabling it removes the guide, aria-describedby relationship, and all enhancement output.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, the complete Risk Class 2 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
