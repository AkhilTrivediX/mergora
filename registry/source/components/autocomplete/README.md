# Autocomplete

Accessible combobox suggestions with Mergora's literal-white work surface, ink structure, bounded corners, and shared focus treatment.

## Mergora advantage

`showMatchContext` adds a visible polite count and active-result position. It defaults off; disabling it removes the output, description relationship, and live-region behavior while the baseline combobox remains unchanged.

Consumer-owned remote collections can expose `loading`, `loadError`/`onRetry`, and `hasMore`/`onLoadMore` independently. These props render bounded busy and recovery affordances but never initiate network work.

## Status

`source-present-unreleased`. Generated parity, packed consumers, lifecycle fixtures, full browser and manual assistive-technology evidence, public dogfooding, and an approved Quality Passport remain blockers.
