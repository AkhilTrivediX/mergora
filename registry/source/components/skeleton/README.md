# Skeleton canonical source

Status: source present and unreleased. Automated workbench evidence is not release evidence; no Stable or conformance claim is made here.

Skeleton is an accessibility-hidden placeholder that reserves the logical size of future content.

## Mergora signature and advantage

Skeleton uses the family's restrained radius, precise Ink-like boundary, Surface fill, and purposeful opacity motion rather than an unstructured animated block. Beyond a baseline Shadcn Skeleton, logical dimensions and shapes preserve the future footprint and `animated` independently controls motion. `animated={false}` removes the animation hook and all pulse behavior while retaining identical geometry and the same `aria-hidden` contract.

## Contract

- It accepts no role, live-region attribute, accessible label, children, or fake text. TypeScript omits these props and the runtime rejects semantic overrides from JavaScript callers. The owning content region supplies aria-busy.
- Text, rectangle, and circle shapes use explicit logical block and inline dimensions; numeric values normalize to CSS pixels.
- `animated=true` preserves the existing pulse; `animated=false` removes it explicitly, reduced motion disables it automatically, and forced colors retains a boundary.

The public ref and stable data-slot parts are recorded in skeleton.anatomy.json. The exact five-key source manifest, metadata, API, story-state policy, accessibility contract, and honest promotion delta live beside the implementation.
