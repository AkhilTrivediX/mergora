# Emoji Picker

Status: source present and unreleased. No Stable, complete Unicode/CLDR dataset, virtualization, package-parity, or manual Risk Class 3 claim is made.

`EmojiPicker` provides a real row-owned ARIA grid with one roving tab stop, complete spatial arrows, row and collection bounds, RTL mirroring, visible localized labels, and touch-sized buttons. Search and category filters support controlled or uncontrolled state; consumer-owned recent ordering, skin-tone selection, and filtered-result announcements remain independent options. Disabling one removes its controls, callbacks, output, and accessibility context. Mergora stores nothing and sends no query externally.

Consumers must supply a licensed and versioned localized dataset, decide recency privacy/retention, and provide an accessible measured windowing strategy for very large collections. This checkpoint does not pretend its small fixture is Unicode coverage or virtualization proof.

The literal Canvas collection, Ink rails, strong control labels, Green selection, restrained Violet recent cue, shared focus geometry, bounded cells, logical layout, narrow stacking, forced-color Highlight mapping, and no animated reordering keep the picker coherent and calm.
