# Dropdown Menu

A data-driven React Aria menu with Mergora's literal-white surface, precise Ink dividers, strong label/context hierarchy, Green selection signal, Violet focus seam, restrained corners, and preference-safe overlay motion.

## Mergora advantage

Optional live selection context and a two-step destructive-action confirmation make dense action menus safer without changing the basic menu or forcing either aid. Selection summary and destructive confirmation are separate booleans. Each false value removes its UI, state branch, events, and accessibility output independently.
