# Calendar Heatmap

Native-first Calendar Heatmap source with Mergora's literal-white surface, ink structure, restrained green/violet state signals, bounded corners, and forced-colors treatment.

## Mergora advantage

An optional aggregate and selection summary complements the always-available keyboard grid, non-color marks, legend, and table fallback. Omitting or disabling the documented enhancement removes its UI, behavior, callbacks, and accessibility output while the baseline remains usable.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, supported-browser evidence, visual review, the complete Risk Class 3 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
