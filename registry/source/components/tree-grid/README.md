# TreeGrid canonical source

Status: source present and unreleased. No Stable, package-parity, large-data performance, APG interoperability-complete, or manual assistive-technology claim is made.

`TreeGrid` exposes a one-tab-stop ARIA treegrid with hierarchical row metadata, controlled or uncontrolled expansion/selection/active row, direction-aware two-dimensional keyboard movement, selection, native disclosure buttons, optional async editing with recovery, form serialization/reset, and a consumer-controlled virtual window that preserves real positions.

The optional hierarchy summary, localized change announcements, editing, and virtualization are independent Mergora enhancements. Disabling each removes its UI, behavior, event path, and accessibility output. Literal Canvas cells, Ink rules, Green selection/status, Violet active/focus seams, restrained corners, logical indentation, visible narrow overflow, forced colors, and reduced-motion fallbacks provide the family identity.

Promotion still requires regenerated parity, packed consumers, complete APG/browser interoperability, 10k-row performance evidence, and current Risk Class 3 manual desktop/mobile screen-reader, speech, switch, keyboard, touch, zoom, RTL, and forced-colors sessions.
