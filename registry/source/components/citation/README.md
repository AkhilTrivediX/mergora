# Citation

`Citation` provides stable numbering, a precise accessible link name and a fail-closed URL policy. It does not fetch or attest to source content.

## Mergora advantage

`showSourceDetail` adds a directly reachable named note while the baseline stays a concise citation. `false` removes that note from layout and the accessibility tree. Unsafe URLs never become links.

Status: source present and unreleased. Full parity, browser/security, packed-consumer, and manual assistive-technology evidence remain blockers.
