# PromptSuggestions

`PromptSuggestions` renders domain-neutral prompt actions as native buttons. Selection semantics are never applied merely for styling.

## Mergora advantage

`selectionMode` atomically opts into controlled/uncontrolled selection, listbox/option semantics and roving keyboard focus. `showDescriptions` is separate. Disabling either removes its own UI, behavior and accessibility output.

Status: source present and unreleased. Full distribution parity, browser/localization evidence, packed consumers, and the manual assistive-technology matrix remain blockers.
