# QueryBuilder canonical source

Status: source present and unreleased. No Stable, package-parity, performance-scale, or manual assistive-technology claim is made.

`QueryBuilder` owns nested fieldset/legend condition groups, schema-bound field/operator/value editors, controlled or uncontrolled state, deterministic JSON serialization, form reset, built-in and consumer validation, localized messages, and explicit Move Up/Down controls. Native controls preserve keyboard, touch, autofill, and zoom behavior without drag dependence.

The optional summary is the Mergora advantage: it turns a nested machine query into a consumer-localizable, decision-safe read-only explanation. `showSummary={false}` removes its UI, renderer call, behavior, and accessibility output. Literal Canvas groups, Ink structure, Green validation/status, Violet nested context, shared focus geometry, restrained corners, logical layout, forced colors, and reduced-motion fallbacks keep it in the family.

Promotion still requires generated distribution parity, packed consumers, large-query performance evidence, and current Risk Class 3 manual desktop/mobile screen-reader, speech, switch, keyboard, touch, zoom, RTL, and forced-colors sessions.
