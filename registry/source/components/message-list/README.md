# MessageList

`MessageList` starts as a labelled scroll region with an ordered list. It does not fetch messages or own model output.

## Mergora advantage

Fixed-height virtualization, user-respecting follow-output, the resume control, and debounced new-message announcements are independent opt-ins. With each set to `false`, its DOM, events, state work, and accessibility output are absent.

Status: source present and unreleased. Full performance, browser, parity, packed-consumer, and current manual assistive-technology evidence remain promotion blockers.
