# Drawer

A Sheet-backed modal drawer with Mergora's literal-white work surface, strong Ink edge, deliberate heading hierarchy, Violet focus seam, bounded corners, and spatial motion removed under reduced-motion.

## Mergora advantage

Optional dependency-free touch swipe dismissal includes a visible keyboard-operable handle, directional start/end logic, and reasoned close details while preserving every baseline close path. Setting swipeToClose adds only the swipe handle, touch pointer handlers, and swipe close reasons. False removes its UI, handlers, events, and accessibility output.
