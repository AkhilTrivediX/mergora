# DateField canonical source

Status: source present and unreleased. No Stable, package-parity, conformance, or manual assistive-technology claim is made.

`DateField` preserves a real native date control for locale editing, platform pickers, validation, serialization, autofill, keyboard operation, and reset. Controlled and uncontrolled values use the browser's canonical date string.

Its optional `showDateContext` enhancement associates a localized full-date reading with the control. The node, relationship, formatting work, and accessibility output are all absent when disabled. Canvas, Ink, Green context rail, Violet focus seam, and semantic density tokens make the field part of the Mergora family without adding default scheduling weight.

Promotion still requires regenerated parity, packed consumers, exhaustive locale/platform date-input evidence, and current manual assistive-technology records.
