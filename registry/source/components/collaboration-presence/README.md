# CollaborationPresence

`CollaborationPresence` shows a labelled native list with person names and explicit available, away, busy, stale, or offline text. It owns no real-time connection.

## Mergora advantage

An explicit-clock `stalePolicy` derives stale state without SSR/hydration drift; `showSummary` is independent. Both default off and add no UI, work or accessibility output until selected.

Status: source present and unreleased. Full parity, browsers, packed consumers, privacy review, and manual assistive-technology evidence remain blockers.
