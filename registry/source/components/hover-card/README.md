# Hover Card

A focus-, hover-, and press-capable supplemental preview with Mergora's literal-white surface, Ink border, strong context hierarchy, Violet pin rail, restrained corners, and preference-safe motion.

## Mergora advantage

Optional press-to-pin gives keyboard, touch, and pointer users a persistent inspectable preview with an explicit status rail and close path. pinOnPress false removes pinned state, pin status, close control, pin-specific events, and accessibility output while hover/focus preview behavior remains.

## Status

Current status is `source-present-unreleased`. Generated parity, packed consumers, the complete Risk Class 2 manual matrix, public dogfooding, and an approved Quality Passport remain promotion blockers.
